Списки

Списки — один из самых часто используемых элементов в HTML. Они помогают структурировать информацию. В HTML есть три вида списков: маркированные, нумерованные и списки определений.

Урок 1 — Маркированные списки <ul>

Тег <ul> (unordered list — неупорядоченный список) создаёт маркированный список. Каждый элемент списка оборачивается в тег <li> (list item). По умолчанию браузер отображает маркеры в виде точек (•).

Урок 2 — Нумерованные списки <ol>

Тег <ol> (ordered list — упорядоченный список) создаёт нумерованный список. Браузер автоматически нумерует элементы. Атрибут start позволяет начать нумерацию с другого числа. Атрибут reversed делает нумерацию в обратном порядке.

Урок 4 — Вложенные списки

Списки можно вкладывать друг в друга. Для этого нужно поместить новый список <ul> или <ol> внутрь элемента <li>. Такой подход используется для создания многоуровневых меню и иерархических данных.

Урок 5 — Списки определений <dl>

Тег <dl> (description list) создаёт список определений или пар "термин–описание". <dt> — термин (definition term), <dd> — описание (definition description). Используется для словарей, FAQ, глоссариев.
