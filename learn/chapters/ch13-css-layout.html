CSS Layout

Layout (компоновка) — это расположение элементов на странице. CSS предоставляет несколько механизмов компоновки: display, position, float. Понимание этих свойств — основа вёрстки веб-страниц.

Урок 1 — Display

Свойство display определяет, как элемент отображается в потоке документа.

ЗначениеОписаниеПримеры тегов
blockЗанимает всю ширину, новая строка до и после<div>, <p>, <h1>
inlineТолько нужная ширина, не разрывает строку<span>, <a>, <strong>
inline-blockКак inline, но можно задать ширину/высоту—
noneЭлемент скрыт и не занимает место—
flexFlexbox контейнер—
gridGrid контейнер—

Урок 2 — Position

Свойство position управляет позиционированием элементов. Совместно используются свойства top, right, bottom, left.

ЗначениеОписание
staticПо умолчанию — элемент в обычном потоке
relativeСмещение от нормальной позиции
absoluteОтносительно ближайшего positioned родителя
fixedОтносительно окна браузера, не скроллится
stickyПрилипает при прокрутке

Урок 3 — Overflow и visibility

Свойство overflow управляет поведением содержимого, которое выходит за пределы элемента. visibility скрывает элемент, но сохраняет его место.

Урок 4 — Z-index

Свойство z-index управляет порядком наложения positioned-элементов. Чем больше значение, тем выше элемент на «стопке». Работает только с элементами, у которых position ≠ static.

Урок 5 — Float и clearfix

Свойство float выносит элемент в сторону, позволяя тексту обтекать его. Ранее использовалось для компоновки, сейчас чаще применяется для обтекания изображений. clear отменяет обтекание.
