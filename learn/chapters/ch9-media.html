Медиа

HTML5 предоставляет нативную поддержку аудио, видео и графики прямо в браузере — без Flash и других плагинов. Вы научитесь встраивать медиа-контент, работать с canvas и SVG, а также использовать iframe для embed-контента.

Урок 1 — Аудио <audio>

Тег <audio> встраивает аудиофайл. Атрибут controls отображает встроенный плеер с кнопками управления. Атрибут autoplay запускает воспроизведение автоматически (в большинстве браузеров требует также muted). Используйте несколько тегов <source> для поддержки разных форматов.

Урок 2 — Видео <video>

Тег <video> аналогичен <audio>, но для видеофайлов. Атрибуты width и height задают размер видеоплеера. Атрибут poster указывает изображение, которое показывается до начала воспроизведения.

Урок 3 — Встраивание <iframe>

Тег <iframe> встраивает другую HTML-страницу или внешний контент (YouTube, Google Maps, Figma и т.д.) прямо на вашу страницу. Атрибут src указывает URL встраиваемой страницы. Атрибут allowfullscreen разрешает полноэкранный режим для видео.

⚠️
Безопасность: Не все сайты разрешают встраивание через iframe (устанавливают заголовок X-Frame-Options). Для YouTube используйте специальные embed-ссылки (youtube.com/embed/VIDEO_ID).

Урок 4 — Canvas — рисование через JavaScript

Тег <canvas> создаёт холст для рисования через JavaScript. Сам по себе тег — это просто прямоугольная область. Рисование выполняется с помощью JavaScript API. Используется для игр, анимаций, диаграмм.

Урок 5 — SVG — векторная графика

SVG (Scalable Vector Graphics) — это XML-формат для векторной графики. SVG-изображения масштабируются без потери качества. Тег <svg> позволяет встраивать SVG прямо в HTML. SVG поддерживает фигуры, текст, градиенты и анимации.

Сделайте сами — Упражнение 9

Встройте YouTube видео с iframe

Задание: Создайте страницу с встроенным YouTube видео:
  • Добавьте тег <iframe>
  • Укажите атрибут src с ссылкой на YouTube видео (формат: https://www.youtube.com/embed/ID_ВИДЕО)
  • Задайте размеры через атрибуты width и height
  • Ссылка должна содержать youtube.com или youtu.be

Подсказка: для видео https://www.youtube.com/watch?v=dQw4w9WgXcQ embed-ссылка будет: https://www.youtube.com/embed/dQw4w9WgXcQ
