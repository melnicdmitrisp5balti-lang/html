Адаптивный дизайн

Адаптивный (responsive) дизайн — подход, при котором сайт выглядит корректно на любом устройстве: смартфоне, планшете или компьютере. Ключевые инструменты — viewport meta-тег, media queries и гибкие единицы измерения.

Урок 1 — Что такое адаптивный дизайн?

Более половины интернет-трафика поступает с мобильных устройств. Адаптивный дизайн означает, что сайт автоматически подстраивается под размер экрана. Три столпа адаптивного дизайна:

ПринципОписание
Гибкая сетка Использование относительных единиц (%, fr) вместо фиксированных (px)
Гибкие изображения max-width: 100% — изображение никогда не шире своего контейнера
Media Queries Разные стили для разных размеров экрана
📱
Mobile First: Рекомендуется сначала проектировать для мобильных устройств, затем постепенно добавлять стили для больших экранов. Это делает CSS проще и производительнее.

Урок 2 — Viewport и meta тег

Без viewport meta-тега мобильный браузер масштабирует страницу, как если бы она была для десктопа. Мета-тег viewport сообщает браузеру, как обрабатывать ширину страницы.

💡
Всегда добавляйте в <head>:
<meta name="viewport" content="width=device-width, initial-scale=1.0">

width=device-width — ширина равна ширине экрана устройства.
initial-scale=1.0 — начальный масштаб 100% (без зума).

Урок 3 — Media Queries

Media Queries позволяют применять CSS только при определённых условиях (ширина экрана, ориентация и т.д.). Синтаксис: @media (условие) { стили }.

Точка разрыва (breakpoint)Устройство
max-width: 480pxСмартфон (маленький)
max-width: 768pxПланшет / смартфон
max-width: 1024pxНебольшой ноутбук
min-width: 1200pxБольшой монитор

Урок 4 — Адаптивная типографика

Для адаптивной типографики используют относительные единицы и функцию clamp().

ЕдиницаОписание
emОтносительно размера шрифта родителя
remОтносительно размера шрифта корневого элемента (html)
vw1% ширины viewport
vh1% высоты viewport
clamp(min, ideal, max)Размер между min и max, идеально — ideal
