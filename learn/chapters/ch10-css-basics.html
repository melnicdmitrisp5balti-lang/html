Основы CSS

CSS (Cascading Style Sheets — Каскадные таблицы стилей) — язык, который отвечает за внешний вид HTML-страниц: цвета, шрифты, расположение элементов. В этой главе вы узнаете, как работает CSS, как его подключать и как писать правила стилей.

Урок 1 — Что такое CSS?

CSS описывает, как должны выглядеть HTML-элементы. Без CSS страница выглядит как простой текст на белом фоне. CSS добавляет цвета, шрифты, отступы и многое другое.

💡
HTML vs CSS: HTML — это структура (что показывать), CSS — это стиль (как это выглядит). Разделение структуры и стиля — основа современной веб-разработки.

Урок 2 — Способы подключения CSS

Существует три способа добавить CSS на страницу. Каждый имеет своё применение:

СпособСинтаксисГде использовать
Внутренний (inline) <p style="color:red"> Для одного элемента
Внутристраничный <style>...</style> в <head> Для одной страницы
Внешний (рекомендуется) <link rel="stylesheet" href="style.css"> Для всего сайта

Урок 3 — Селекторы CSS

Селектор указывает, к каким HTML-элементам применить стиль. Существуют разные виды селекторов:

СелекторПримерЧто выбирает
Элементp { }Все теги <p>
Класс.title { }Элементы с class="title"
ID#header { }Элемент с id="header"
Универсальный* { }Все элементы
Группаh1, h2 { }Все h1 и h2
Потомокdiv p { }<p> внутри <div>

Урок 5 — Каскадность и специфичность

«Каскадные» в названии CSS означает, что правила применяются в определённом порядке. Если к одному элементу применяются несколько правил, побеждает то, которое имеет большую специфичность (приоритет).

Вид селектораВес (специфичность)
* { } — универсальный0
p { } — элемент1
.class { } — класс10
#id { } — идентификатор100
style="..." — inline1000
!importantВысший приоритет

Напишите CSS-стили для HTML-страницы

Задание: Создайте HTML-страницу и оформите её с помощью CSS. Страница должна содержать:
  • Тег <style> с CSS-правилами
  • Стиль для элемента (например body или h1) с цветом текста
  • Стиль через класс (.имя) с фоновым цветом
  • Стиль через ID (#имя)
