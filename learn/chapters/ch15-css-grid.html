CSS Grid

CSS Grid — двумерная система компоновки, позволяющая работать одновременно со строками и столбцами. Grid идеально подходит для создания сложных макетов страниц, фотогалерей и панелей управления.

Урок 1 — Что такое CSS Grid?

Grid включается свойством display: grid. Сетка определяется колонками (grid-template-columns) и строками (grid-template-rows).

🔲
Grid vs Flexbox: Flexbox — одномерный (строки ИЛИ столбцы). Grid — двумерный (строки И столбцы одновременно). Используйте Grid для макетов страниц, Flexbox — для компонентов внутри.

Урок 2 — Колонки и строки

grid-template-columns задаёт ширину каждой колонки. Единица fr (fraction) — доля свободного пространства. Функция repeat() помогает не повторять одинаковые значения. minmax() задаёт минимальный и максимальный размер трека.

Урок 3 — Промежутки и выравнивание

Свойство gap (или column-gap / row-gap) задаёт промежутки между ячейками. justify-items и align-items выравнивают содержимое ячеек по горизонтали и вертикали соответственно.

Урок 4 — Размещение элементов

С помощью grid-column и grid-row можно точно разместить элемент в сетке и растянуть его на несколько ячеек. Синтаксис: grid-column: начало / конец или span N.

grid-template-areas позволяет задавать макет страницы с помощью наглядных имён. Каждый элемент получает имя через grid-area.

Создайте сетку карточек с CSS Grid

Задание: Создайте фотогалерею или сетку карточек с помощью CSS Grid. Используйте:

  • Свойство gap для промежутков
  • Хотя бы один элемент с grid-column: span 2 или grid-row: span 2
