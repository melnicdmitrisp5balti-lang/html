Блочная модель CSS

Каждый HTML-элемент — это прямоугольный блок. Блочная модель (box model) описывает, из чего состоит этот блок: содержимое, внутренние отступы (padding), рамка (border) и внешние отступы (margin). Понимание box model — ключ к правильной вёрстке.

Урок 1 — Что такое блочная модель?

Блочная модель CSS состоит из четырёх слоёв (снаружи внутрь): margin → border → padding → content.

📦
Слои блока: margin — внешний отступ (прозрачный, отделяет элемент от других), border — видимая рамка, padding — внутренний отступ (между рамкой и содержимым), content — само содержимое (текст, картинки).

Урок 2 — Padding (внутренние отступы)

Свойство padding задаёт расстояние между содержимым и рамкой. Можно задавать отступы для каждой стороны отдельно.

СвойствоОписание
padding: 20pxВсе четыре стороны — 20px
padding: 10px 20pxВерх/низ 10px, право/лево 20px
padding: 5px 10px 15px 20pxВерх, право, низ, лево
padding-top: 10pxТолько верхний отступ
padding-right: 10pxТолько правый отступ
padding-bottom: 10pxТолько нижний отступ
padding-left: 10pxТолько левый отступ

Урок 3 — Margin (внешние отступы)

Свойство margin задаёт пространство снаружи элемента — между ним и соседними элементами. Синтаксис аналогичен padding. Особый случай: margin: 0 auto — горизонтальное центрирование блока.

Урок 4 — Border (рамка)

Свойство border задаёт рамку вокруг элемента. Оно принимает три значения: толщину, стиль и цвет.

Стиль рамкиОписание
solidСплошная линия
dashedПунктирная
dottedТочечная
doubleДвойная линия
noneБез рамки

Урок 5 — Width, Height и box-sizing

По умолчанию width и height задают размер только содержимого. Padding и border добавляются поверх, что делает элемент больше ожидаемого. Свойство box-sizing: border-box включает padding и border в ширину элемента.

✅
Лучшая практика: В начале CSS-файла пишут *, *::before, *::after { box-sizing: border-box; } — это упрощает расчёт размеров всех элементов.

Сделайте сами — Упражнение 11

Создайте карточку с помощью блочной модели

Задание: Создайте «карточку товара» с CSS-стилями. Карточка должна содержать:
  • Свойство padding (внутренний отступ)
  • Свойство margin (внешний отступ)
  • Свойство border (рамка)
  • Свойство border-radius (скруглённые углы)
  • Свойство width или max-width
