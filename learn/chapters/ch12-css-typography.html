CSS Типографика

Типографика — оформление текста. CSS предоставляет мощные инструменты для управления цветами, шрифтами, размерами, выравниванием и другими свойствами текста. Хорошая типографика делает сайт читаемым и привлекательным.

Урок 1 — Цвет и фон

Свойство color задаёт цвет текста. background-color — цвет фона. Цвет можно указывать разными способами:

Урок 2 — Шрифты (font-family)

Свойство font-family задаёт шрифт. Указывают несколько шрифтов через запятую — браузер использует первый доступный. Последний всегда должен быть общее семейство (serif, sans-serif, monospace).

Урок 4 — Свойства текста

CSS предлагает множество свойств для управления внешним видом текста: выравнивание, отступы, межстрочный интервал, трансформация и другие.

СвойствоОписание
text-alignВыравнивание: left, center, right, justify
text-decorationПодчёркивание: underline, line-through, none
text-transformuppercase, lowercase, capitalize
line-heightМежстрочный интервал: 1.5, 24px
letter-spacingИнтервал между буквами: 2px
word-spacingИнтервал между словами
text-indentОтступ первой строки: 2em
text-shadowТень текста: 2px 2px 4px grey

Урок 5 — Google Fonts

Google Fonts — бесплатный сервис с сотнями красивых шрифтов. Подключите шрифт через <link> в <head> и используйте его в CSS.

🌐
Как подключить: Зайдите на fonts.google.com, выберите шрифт, скопируйте тег <link> и вставьте в <head> вашей страницы.

Сделайте сами — Упражнение 12

Оформите текстовый блок с помощью CSS-типографики

Задание: Создайте страницу с оформленным текстом. Используйте:
  • Свойство color для цвета текста
  • Свойство background-color для фона
  • Свойство font-size для размера шрифта
  • Свойство font-weight (bold или число)
  • Свойство text-align для выравнивания
  • Свойство line-height для межстрочного интервала
