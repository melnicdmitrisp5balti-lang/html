Формы

Формы позволяют пользователям вводить данные и отправлять их на сервер. Они используются повсюду: регистрация, вход, контактные формы, поиск. Вы научитесь создавать полноценные HTML-формы с различными полями ввода.

Урок 1 — Основы форм <form>

Тег <form> создаёт HTML-форму. Атрибут action указывает URL для отправки данных. Атрибут method задаёт метод HTTP: GET (данные в URL) или POST (данные в теле запроса). Все поля ввода должны быть внутри <form>.

GET vs POST: GET-запрос добавляет данные к URL (видны в строке браузера). POST отправляет данные скрыто. Для форм с паролями всегда используйте метод POST!

Урок 2 — Текстовые поля и метки <label>

Тег <label> создаёт подпись к полю ввода. Связать метку с полем можно двумя способами: через атрибут for (значение должно совпадать с id поля) или вложив поле ввода прямо внутрь тега <label>. Клик по метке автоматически фокусирует связанное поле.

Урок 3 — Выпадающие списки <select>

Тег <select> создаёт выпадающий список. Каждый вариант — это тег <option>. Атрибут value тега <option> — это значение, которое будет отправлено на сервер. Для группировки вариантов используют <optgroup>.

Урок 4 — Многострочное поле <textarea>

Тег <textarea> создаёт многострочное поле для ввода текста. Атрибуты rows и cols задают размер поля в строках и столбцах. Текст по умолчанию помещается между открывающим и закрывающим тегами.

Урок 6 — Кнопки отправки формы

Для отправки формы используют <input type="submit"> или <button type="submit">. Тег <button> гибче — внутрь можно добавлять HTML-контент (иконки, текст). type="reset" сбрасывает форму к начальным значениям.

Форма обратной связи

Задание: Создайте контактную форму. Форма должна содержать:
  • Тег <form>
  • Текстовое поле <input type="text"> для имени
  • Поле <input type="email"> для email
  • Многострочное поле <textarea> для сообщения
  • Метки <label> для каждого поля
  • Кнопку отправки
