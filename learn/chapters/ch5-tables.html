Таблицы

HTML-таблицы позволяют представлять данные в строках и столбцах. Они отлично подходят для расписаний, прайс-листов, сравнительных таблиц и любых структурированных данных.

Урок 1 — Базовая структура таблицы

HTML-таблица создаётся с помощью тега <table>. Строки таблицы задаются тегом <tr> (table row), а ячейки данных — тегом <td> (table data).

Урок 2 — Заголовки таблицы <th>

Тег <th> (table header) создаёт ячейку-заголовок. По умолчанию текст в <th> выводится жирным и выровнен по центру. Заголовки также помогают экранным читалкам понять структуру таблицы. Атрибут scope уточняет, является ли заголовок строчным или столбцовым.

Урок 3 — Подпись к таблице <caption>

Тег <caption> добавляет подпись (заголовок) к таблице. Он должен быть первым элементом внутри <table>. По умолчанию подпись располагается над таблицей. Также можно использовать теги <thead>, <tbody>, <tfoot> для группировки строк.

Урок 4 — Объединение ячеек: colspan и rowspan

Атрибут colspan объединяет ячейки по горизонтали (по столбцам). Атрибут rowspan объединяет ячейки по вертикали (по строкам). Значение указывает, сколько ячеек нужно объединить.

Урок 5 — Стилизация таблиц

Таблицы можно красиво стилизовать с помощью CSS. Свойство border-collapse: collapse убирает двойные границы между ячейками. Чередование цветов строк улучшает читаемость.
