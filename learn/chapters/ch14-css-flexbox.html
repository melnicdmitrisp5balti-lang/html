CSS Flexbox

Flexbox (Flexible Box Layout) — мощный инструмент CSS для создания гибких одномерных макетов (строк или столбцов). С Flexbox легко выравнивать элементы, распределять пространство и создавать адаптивные интерфейсы.

Урок 1 — Что такое Flexbox?

Чтобы включить Flexbox, достаточно написать display: flex на родительском элементе (flex-контейнер). Все его прямые дочерние элементы автоматически становятся flex-элементами и выстраиваются в ряд.

📦
Две оси Flexbox: главная ось (main axis) — направление размещения элементов (по умолчанию горизонтально), поперечная ось (cross axis) — перпендикулярно главной.

Урок 2 — flex-direction и flex-wrap

flex-direction задаёт направление главной оси. flex-wrap управляет переносом элементов на новую строку.

rowГоризонтально слева направо (по умолчанию)
row-reverseГоризонтально справа налево
columnВертикально сверху вниз
column-reverseВертикально снизу вверх

Урок 4 — align-items и align-content

align-items выравнивает элементы по поперечной оси (для row — вертикально). align-content работает при нескольких строках (flex-wrap: wrap).

Урок 5 — Свойства flex-элементов

Свойства, применяемые к самим flex-элементам (не к контейнеру):

СвойствоОписание
flex-growКоэффициент роста (0 = не растёт, 1 = растёт)
flex-shrinkКоэффициент сжатия (по умолчанию 1)
flex-basisБазовый размер до распределения пространства
flexСокращение: flex: grow shrink basis
orderПорядок отображения (по умолчанию 0)
align-selfВыравнивание конкретного элемента по поперечной оси

Создайте навигацию и карточки с помощью Flexbox

Задание: Создайте страницу с навигационным меню и набором карточек. Используйте Flexbox:
  • Свойство display: flex на контейнере
  • Свойство justify-content для распределения элементов
  • Свойство align-items для вертикального выравнивания
  • Свойство gap для отступов между элементами
  • Свойство flex-wrap: wrap для переноса
