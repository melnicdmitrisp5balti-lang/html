Типы Input

HTML5 добавил множество новых типов полей ввода. Каждый тип предоставляет встроенную валидацию и специальный интерфейс (особенно на мобильных устройствах). Узнайте о всех доступных типах input.

Урок 1 — text, password, email, url, search

Базовые текстовые типы полей: text — обычный текст, password — скрывает введённый текст, email — проверяет формат email (содержит @), url — проверяет формат URL, search — поле поиска (с кнопкой очистки в некоторых браузерах).

Урок 2 — number и range

type="number" — числовое поле со стрелками. Атрибуты min, max, step задают диапазон и шаг. type="range" — ползунок для выбора значения в диапазоне.

Урок 3 — date, time, datetime-local, month, week

HTML5 предоставляет удобные поля для выбора даты и времени с нативным виджетом браузера (особенно удобно на мобильных устройствах).

Урок 4 — color и file

type="color" — выбор цвета из палитры (возвращает значение в формате HEX). type="file" — загрузка файла с устройства. Атрибут accept ограничивает типы файлов. Атрибут multiple разрешает загружать несколько файлов.

Урок 5 — Атрибуты валидации

HTML5 позволяет задавать встроенную валидацию без JavaScript. required — поле обязательно для заполнения. minlength/maxlength — ограничение по длине строки. pattern — регулярное выражение для проверки формата.

Урок 6 — Атрибуты полей: readonly, disabled, autocomplete

readonly — поле только для чтения (значение отправляется с формой). disabled — поле отключено (значение не отправляется). autocomplete — управляет автозаполнением браузера. autofocus — поле получает фокус при загрузке страницы.
