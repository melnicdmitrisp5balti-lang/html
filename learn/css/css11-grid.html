CSS Grid

CSS Grid — это двумерная система компоновки, позволяющая размещать элементы одновременно по строкам и столбцам. Это самый мощный инструмент для создания сложных макетов страниц без лишних хаков и костылей. В этом разделе вы освоите все ключевые концепции CSS Grid — от простых сеток до именованных областей и адаптивных раскладок.

Урок 1 — Grid-контейнер

Чтобы включить Grid, нужно задать родительскому элементу display: grid. Затем можно определить количество и размеры колонок с помощью grid-template-columns и строк — grid-template-rows.

СвойствоЗначениеОписание
displaygrid / inline-gridВключает Grid для контейнера
grid-template-columns100px 200px 1frОпределяет колонки
grid-template-rows50px autoОпределяет строки
grid-auto-rowsminmax(100px, auto)Размер автоматически создаваемых строк
💡
Grid vs Flexbox: Flexbox — одномерный (строка или столбец). Grid — двумерный (строки И столбцы одновременно). Используйте Grid для общего макета страницы, Flexbox — для компонентов внутри.

Урок 2 — fr, repeat(), minmax()

Единица fr (fraction) делит доступное пространство пропорционально. Функция repeat() позволяет не дублировать значения, а minmax() задаёт минимальный и максимальный размер трека.

СинтаксисПримерОписание
fr1fr 2fr 1frДробная единица (занимает часть свободного пространства)
repeat(n, size)repeat(3, 1fr)Повторяет значение n раз
minmax(min, max)minmax(100px, 1fr)Размер от min до max
autoautoАвтоматический размер по содержимому
✅
repeat(3, 1fr) эквивалентно 1fr 1fr 1fr — три равные колонки. Сочетайте repeat() и minmax() для мощных адаптивных сеток.

Урок 3 — gap, column-gap, row-gap

Свойство gap задаёт отступы между треками Grid (и Flexbox). Можно указывать одно значение для обоих направлений или два — отдельно для строк и столбцов.

СвойствоПримерОписание
gapgap: 16pxОтступ между строками и столбцами одновременно
row-gaprow-gap: 20pxТолько отступ между строками
column-gapcolumn-gap: 10pxТолько отступ между столбцами
gap (два значения)gap: 20px 10pxПервое — строки, второе — столбцы
💡
Ранее эти свойства назывались grid-gap, grid-row-gap, grid-column-gap — старые названия ещё работают в браузерах, но новые (без префикса grid-) являются стандартом.

Урок 4 — grid-column и grid-row

Элементы Grid можно явно размещать и растягивать по нескольким ячейкам с помощью grid-column и grid-row. Значения задаются через линии сетки: начало / конец или через span N (занять N треков).

СвойствоПримерОписание
grid-column1 / 3С первой по третью вертикальную линию (2 колонки)
grid-columnspan 2Занять 2 колонки
grid-row1 / 3С первой по третью горизонтальную линию (2 строки)
grid-area1 / 1 / 3 / 3row-start / col-start / row-end / col-end
✅
Линии нумеруются с 1 слева направо (для колонок) и сверху вниз (для строк). Можно использовать отрицательные числа для счёта с конца: -1 — последняя линия.

Урок 5 — grid-template-areas

grid-template-areas позволяет задавать макет через понятные текстовые имена областей. Каждый элемент получает имя через grid-area, а контейнер описывает расположение этих имён как ASCII-схему.

СвойствоОписание
grid-template-areasЗадаёт именованные области строками в кавычках
grid-area: nameПрисваивает элементу имя области
. (точка)Пустая ячейка в схеме областей
💡
Имена областей повторяются в нескольких ячейках, если элемент должен их занять. Это позволяет создавать сложные макеты читаемым способом без сложных числовых координат.

Урок 6 — auto-fill и auto-fit

auto-fill и auto-fit вместе с minmax() создают адаптивные сетки без медиазапросов. Колонки сами переносятся на новую строку, когда не помещаются. Разница: auto-fit схлопывает пустые треки, auto-fill оставляет их.

ЗначениеОписание
auto-fillСоздаёт столько треков, сколько поместится (пустые сохраняются)
auto-fitКак auto-fill, но пустые треки схлопываются в 0
minmax(200px, 1fr)Минимум 200px, максимум — заполнить всё
✅
Паттерн repeat(auto-fit, minmax(200px, 1fr)) — стандартный рецепт полностью адаптивной карточной раскладки. Попробуйте изменить размер окна — колонки перестраиваются автоматически!

Создайте полноценный макет страницы с помощью CSS Grid

Задание: Напишите страницу, которая использует CSS Grid для создания полноценного макета:
  • Создать Grid-контейнер с именованными областями: header, sidebar, main и footer через grid-template-areas
  • Header и footer должны занимать всю ширину (обе колонки)
  • Sidebar — фиксированная ширина (около 180px), main — остальное пространство (1fr)
  • Добавить gap между областями и сделать минимальную высоту страницы 300px
  • Добавить хотя бы один элемент внутри main с grid-column: span 2 или несколькими fr-колонками
  • Применить разные фоновые цвета для каждой области для наглядности
