Трансформации

CSS-трансформации позволяют изменять визуальное положение, размер, угол и форму элементов без нарушения потока документа. Трансформации применяются через свойство transform и хорошо сочетаются с переходами и анимациями для создания динамичных интерфейсов.

Урок 1 — translate: перемещение

Функция translate(x, y) перемещает элемент относительно его текущего положения. Это не влияет на поток документа — другие элементы не сдвигаются.

ФункцияПримерОписание
translate(x, y)translate(50px, 20px)Перемещение по X и Y
translateX(n)translateX(100px)Только по горизонтали
translateY(n)translateY(-30px)Только по вертикали
translate(50%)translate(50%, 0)В процентах от размера самого элемента
✅
translate — один из самых производительных способов анимации, так как браузер обрабатывает его через GPU. Предпочитайте translate вместо изменения top/left для анимаций.

Урок 2 — rotate: вращение

Функция rotate(deg) поворачивает элемент на заданный угол. Положительные значения — по часовой стрелке, отрицательные — против. Для 3D-вращения используются rotateX(), rotateY(), rotateZ().

ФункцияПримерОписание
rotate(deg)rotate(45deg)Вращение в плоскости (по часовой)
rotate(-deg)rotate(-90deg)Вращение против часовой
rotateX(deg)rotateX(60deg)Вращение вокруг горизонтальной оси (3D)
rotateY(deg)rotateY(45deg)Вращение вокруг вертикальной оси (3D)
rotateZ(deg)rotateZ(30deg)То же что rotate() — вокруг оси Z
💡
Градусы можно заменить на turn (обороты): rotate(0.5turn) = rotate(180deg), или rad (радианы). В большинстве случаев удобнее использовать deg.

Урок 3 — scale: масштабирование

Функция scale(x, y) изменяет размер элемента. Значение 1 — исходный размер, 2 — вдвое больше, 0.5 — вдвое меньше. Отрицательные значения зеркалят элемент.

ФункцияПримерОписание
scale(n)scale(1.5)Увеличить в 1.5 раза по обеим осям
scale(x, y)scale(2, 0.5)По-разному по X и Y
scaleX(n)scaleX(2)Только по горизонтали
scaleY(n)scaleY(0.5)Только по вертикали
scale(-1)scale(-1, 1)Зеркальное отражение по X
✅
scale не влияет на соседние элементы в потоке — элемент как бы «растёт над страницей». Это отличает scale от изменения width/height.

Урок 4 — skew: наклон

Функция skew(xdeg, ydeg) наклоняет элемент по горизонтали и вертикали, создавая эффект скоса. Используется для декоративных эффектов и создания параллелограммов.

ФункцияПримерОписание
skew(x, y)skew(20deg, 10deg)Наклон по X и Y
skewX(deg)skewX(30deg)Только по горизонтали
skewY(deg)skewY(15deg)Только по вертикали
💡
skew может сильно исказить текст. Для декоративных скошенных блоков часто лучше использовать skewX() на обёртке и обратный skewX() на содержимом, чтобы текст оставался прямым.

Урок 5 — transform-origin и комбинации

transform-origin определяет точку, вокруг которой выполняется трансформация. По умолчанию — центр элемента (50% 50%). Несколько трансформаций можно записать через пробел в одном transform.

ЗначениеОписание
50% 50% (умолч.)Центр элемента
0 0Верхний левый угол
100% 100%Нижний правый угол
top leftКлючевые слова: top/center/bottom + left/center/right
✅
При комбинировании трансформаций порядок важен: translate(50px) rotate(45deg) ≠ rotate(45deg) translate(50px). Трансформации применяются справа налево.

Урок 6 — 3D-трансформации

CSS поддерживает 3D-трансформации. Свойство perspective задаёт глубину проекции (на родителе), а rotateX()/rotateY()/rotateZ() вращают вокруг соответствующих осей в 3D-пространстве.

Свойство/функцияПримерОписание
perspectiveperspective: 600pxГлубина 3D-перспективы (на контейнере)
rotateX(deg)rotateX(60deg)Вращение вокруг горизонтальной оси
rotateY(deg)rotateY(45deg)Вращение вокруг вертикальной оси
transform-styletransform-style: preserve-3dДочерние элементы в 3D-пространстве
backface-visibilitybackface-visibility: hiddenСкрыть обратную сторону элемента
💡
Для полного переворота карточки (card flip) используйте transform-style: preserve-3d на контейнере и два дочерних элемента (лицо и обратная сторона) с rotateY(180deg) на обратной стороне.

Создайте эффект переворота карточки с помощью трансформаций

Задание: Напишите CSS-карточку, которая переворачивается при наведении (card flip):
  • Создать контейнер .card с perspective и transform-style: preserve-3d для 3D-эффекта
  • Добавить два дочерних элемента .front и .back — лицевая и обратная стороны карточки
  • Задать .back начальный rotateY(180deg) и обоим элементам backface-visibility: hidden
  • При :hover на карточке — повернуть всю карточку на rotateY(180deg) с плавным transition
  • Использовать разные фоновые цвета или изображения для лицевой и обратной сторон
  • Добавить любой текст или иконку на обе стороны карточки
