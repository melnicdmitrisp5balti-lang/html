Цвета и фоны

Цвета и фоны — это основа визуальной привлекательности веб-страниц. CSS предоставляет множество способов задавать цвета: именованные цвета, HEX, RGB, HSL — и каждый из них удобен в разных ситуациях. Фоновые свойства позволяют добавлять не только цвет, но и изображения, градиенты и сложные многослойные композиции.

Урок 1 — Форматы цветов

CSS поддерживает несколько форматов записи цвета. Выбор формата зависит от задачи: именованные — для стандартных цветов, HEX — для точного дизайна, HSL — для удобного создания цветовых схем, RGBA/HSLA — когда нужна прозрачность.

ФорматПримерОсобенность
Именованныйcolor: tomato140+ стандартных имён
HEXcolor: #e74c3cКомпактно, широко используется
HEX с прозрачностьюcolor: #e74c3cccПоследние 2 цифры — альфа-канал
RGBcolor: rgb(231, 76, 60)Красный, Зелёный, Синий (0–255)
RGBAcolor: rgba(231,76,60, 0.5)+ альфа-канал (0–1)
HSLcolor: hsl(6, 78%, 57%)Тон, Насыщенность, Светлота
HSLAcolor: hsla(6, 78%, 57%, 0.5)+ альфа-канал
✅
HSL — для дизайнеров: В формате HSL легко создавать оттенки одного цвета — меняйте только светлоту (L): hsl(200, 70%, 30%) — тёмный, hsl(200, 70%, 60%) — средний, hsl(200, 70%, 85%) — светлый.

Урок 2 — Свойства фона

CSS предоставляет целый набор свойств для управления фоном элемента. Помимо цвета, можно задавать фоновое изображение, управлять его размером, позицией и повторением.

СвойствоЗначенияОписание
background-colorцветЦвет фона
background-imageurl(), noneФоновое изображение
background-sizecover, contain, px/%Размер изображения
background-positioncenter, top left, px/%Позиция изображения
background-repeatno-repeat, repeat-x/yПовторение фона
backgroundсокращениеВсе свойства в одном
💡
cover vs contain: cover — изображение заполняет весь блок (может обрезаться). contain — изображение помещается целиком (могут быть пустые полосы).

Урок 3 — Градиенты

CSS-градиенты — это плавный переход между двумя и более цветами. Они создаются функциями linear-gradient() и radial-gradient() и используются как значение background-image.

✅
Направление в linear-gradient: to right, to bottom right, или угол в градусах: 45deg, 135deg. Угол 0deg — снизу вверх, 90deg — слева направо.

Урок 4 — Множественные фоны

CSS позволяет задавать несколько фоновых слоёв через запятую. Первый слой в списке — самый верхний. Свойство background-attachment управляет поведением фона при прокрутке страницы.

background-attachmentОписание
scrollФон прокручивается вместе с элементом (по умолчанию)
fixedФон фиксируется относительно окна браузера (параллакс-эффект)
localФон прокручивается вместе с содержимым элемента
💡
Слои фона: Порядок слоёв в background через запятую соответствует их порядку в каждом отдельном свойстве. Первый слой — сверху, последний — снизу. background-color всегда под всеми слоями.

Урок 5 — opacity vs rgba прозрачность

Прозрачность в CSS можно задать двумя способами: через свойство opacity или через цвет с альфа-каналом (rgba / hsla). Разница критически важна: opacity делает прозрачным весь элемент включая потомков, тогда как rgba влияет только на цвет фона/текста.

⚠️
Ловушка opacity: При opacity: 0.5 на родительском элементе все дочерние элементы тоже становятся полупрозрачными — и это нельзя отменить у потомков. Если нужен прозрачный фон, но непрозрачный текст — используйте background: rgba(...).
