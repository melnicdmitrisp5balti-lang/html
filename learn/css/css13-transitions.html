Переходы (Transitions)

CSS-переходы позволяют плавно анимировать изменения свойств без JavaScript. Достаточно указать, какое свойство анимировать, как долго и по какой кривой времени — и браузер автоматически создаст плавный эффект между двумя состояниями элемента.

Урок 1 — transition-property, duration, delay

Три ключевых свойства перехода: transition-property задаёт, что анимировать, transition-duration — длительность, transition-delay — задержку перед стартом.

СвойствоПримерОписание
transition-propertybackgroundКакое CSS-свойство анимировать
transition-duration0.5sДлительность перехода
transition-delay0.2sЗадержка перед началом перехода
💡
Совет: transition-property: all анимирует все изменяемые свойства, но лучше указывать конкретные для производительности.

Урок 2 — transition-timing-function

transition-timing-function определяет скорость изменения свойства во времени — ускорение, замедление или постоянная скорость.

ЗначениеОписание
easeМедленно → быстро → медленно (по умолчанию)
linearПостоянная скорость
ease-inМедленный старт, быстрый финиш
ease-outБыстрый старт, медленный финиш
ease-in-outМедленный старт и финиш
cubic-bezier(...)Полный контроль через кривую Безье
✅
Совет: ease — стандарт для большинства интерфейсов. linear — для непрерывных анимаций (спиннер). cubic-bezier даёт полный контроль.

Урок 3 — Shorthand и несколько свойств

Сокращённое свойство transition объединяет все четыре свойства в одну строку. Несколько переходов перечисляются через запятую.

СинтаксисОписание
transition: property duration timing-function delayПорядок shorthand
transition: color 0.3s, background 0.5s, transform 0.2sНесколько свойств через запятую
💡
Несколько свойств: Для анимации нескольких свойств перечислите их через запятую: transition: color 0.3s, background 0.5s, transform 0.2s

Сделайте сами — Упражнение 13

Создайте интерактивный набор кнопок с transition-эффектами

Задание: Напишите HTML-страницу с CSS-стилями, которая демонстрирует работу CSS-переходов:
  • Создать минимум 4 кнопки с разными transition-эффектами (цвет, трансформация, тень, граница)
  • Каждая кнопка должна использовать разную transition-timing-function
  • Добавить transition-delay хотя бы для одной кнопки
  • Использовать shorthand запись transition с несколькими свойствами
  • Кнопки должны плавно возвращаться в исходное состояние при убирании курсора
