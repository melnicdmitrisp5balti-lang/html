Фильтры и эффекты

CSS-фильтры позволяют применять графические эффекты к элементам прямо в браузере: размытие, изменение яркости, контрастности, оттенков. Также CSS поддерживает backdrop-filter для стеклянного эффекта и mix-blend-mode для наложения слоёв.

Урок 1 — blur, brightness, contrast, grayscale

Основные функции filter для изменения внешнего вида элементов.

ФункцияПримерОписание
blur()filter: blur(4px)Размытие элемента
brightness()filter: brightness(1.5)Яркость (>1 — светлее)
contrast()filter: contrast(200%)Контрастность
grayscale()filter: grayscale(100%)Чёрно-белый
💡
blur(): filter: blur() сглаживает края элемента. Для производительности предпочтительно применять blur только к фоновым/декоративным элементам.

Урок 2 — hue-rotate, invert, saturate, sepia, opacity

Дополнительные фильтры для управления цветом и прозрачностью элементов.

ФункцияПримерОписание
hue-rotate(deg)filter: hue-rotate(180deg)Сдвиг цветового тона
invert(%)filter: invert(100%)Инверсия цветов
saturate(%)filter: saturate(300%)Насыщенность
sepia(%)filter: sepia(100%)Эффект сепии
opacity(%)filter: opacity(50%)Прозрачность
✅
filter: opacity(): Работает как свойство opacity, но может быть аппаратно ускорен. Для производительной анимации прозрачности предпочтителен именно filter: opacity().

Урок 4 — backdrop-filter (glassmorphism)

backdrop-filter применяет фильтр к содержимому за элементом, создавая эффект матового стекла.

ФункцияОписание
backdrop-filter: blur(px)Размытие фона за элементом
backdrop-filter: brightness(%)Яркость фона за элементом
backdrop-filter: saturate(%)Насыщенность фона за элементом
✅
Glassmorphism: полупрозрачный фон (rgba) + backdrop-filter: blur(10px) + тонкая граница rgba(255,255,255,0.3). Добавьте яркий фон позади для лучшего эффекта.

Урок 5 — mix-blend-mode

mix-blend-mode определяет, как цвета элемента смешиваются с фоном под ним.

ЗначениеОписание
normalБез смешивания (по умолчанию)
multiplyУмножение — затемняет
screenЭкран — осветляет
overlayНаложение — контраст
differenceРазность — инвертирует разницу
color-dodgeОсветление цвета
hueТон верхнего слоя
💡
mix-blend-mode: определяет как цвет элемента смешивается с фоном. screen — осветляет, multiply — затемняет, difference — инвертирует разницу.

Сделайте сами — Упражнение 15

Создайте галерею изображений с hover-фильтрами

Задание: Напишите HTML-страницу с CSS-фильтрами:
  • Создать сетку из минимум 6 «карточек» (div с gradient-фоном) через CSS Grid или Flexbox
  • В исходном состоянии применить filter: grayscale(80%) brightness(0.8) к каждой карточке
  • При наведении убрать фильтры через filter: none с transition: filter 0.4s ease
  • Добавить transform: scale(1.05) на hover для интерактивности
  • Реализовать одну карточку с backdrop-filter для glassmorphism эффекта
  • Использовать разные gradient-фоны для каждой карточки
