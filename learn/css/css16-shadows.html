Тени

Тени — важный инструмент дизайна. CSS предоставляет box-shadow для теней блоков, text-shadow для теней текста и filter: drop-shadow() для теней произвольных фигур. Тени добавляют глубину и объём интерфейсу.

Урок 1 — box-shadow: основы

Свойство box-shadow добавляет тень к блочному элементу. Синтаксис: box-shadow: offset-x offset-y blur spread color;

ПараметрОписаниеПример
offset-xСмещение по горизонтали4px
offset-yСмещение по вертикали4px
blurРазмытие (чем больше — тем мягче)10px
spreadРазмер тени (опционально)2px
colorЦвет тениrgba(0,0,0,0.3)

Урок 2 — Множественные тени и inset

Можно задать несколько теней через запятую. Ключевое слово inset делает тень внутренней (вдавленной).

✅
Тени перечисляются через запятую. Первая тень — самая верхняя (ближайшая к элементу).

Урок 3 — text-shadow

Свойство text-shadow добавляет тень к тексту. Синтаксис такой же: text-shadow: offset-x offset-y blur color; (без параметра spread).

Урок 4 — filter: drop-shadow()

filter: drop-shadow() — это фильтр, который добавляет тень к реальной форме элемента (включая прозрачные области PNG-изображений), в отличие от box-shadow, который следует прямоугольнику.

💡
box-shadow vs drop-shadow: box-shadow добавляет тень к прямоугольнику, drop-shadow — к реальному контуру элемента (например, треугольнику из clip-path).

Создайте карточку с красивыми тенями

Задание: Создайте стильную карточку товара с использованием теней.
  • Используйте box-shadow с размытием не менее 15px
