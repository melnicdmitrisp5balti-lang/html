Flexbox

Flexbox (Flexible Box Layout) — мощная модель раскладки CSS для создания гибких одномерных интерфейсов. Она решает классические проблемы вёрстки: горизонтальное и вертикальное выравнивание, равномерное распределение пространства, адаптивные сетки карточек. Flexbox стал стандартом современной веб-разработки.

Урок 1 — flex container: flex-direction

Добавив display: flex к контейнеру, вы создаёте flex-контейнер. Его дочерние элементы становятся flex-элементами и по умолчанию выстраиваются в горизонтальный ряд. flex-direction управляет главной осью: row — горизонтально, column — вертикально.

ЗначениеНаправлениеОписание
row→Слева направо (по умолчанию)
row-reverse←Справа налево
column↓Сверху вниз
column-reverse↑Снизу вверх

Урок 2 — justify-content

justify-content управляет распределением flex-элементов по главной оси (горизонтально для row, вертикально для column). Это свойство решает вопрос «куда девать свободное пространство».

ЗначениеПоведение
flex-startЭлементы прижаты к началу (по умолчанию)
flex-endЭлементы прижаты к концу
centerЭлементы по центру
space-betweenРавные пробелы между элементами, без крайних
space-aroundРавные пробелы вокруг каждого элемента
space-evenlyАбсолютно равные пробелы везде

Урок 3 — align-items

align-items управляет выравниванием flex-элементов по поперечной оси (вертикально для row). Это ключевое свойство для вертикального центрирования, которое раньше было сложной задачей в CSS.

ЗначениеПоведение
stretchРастянуть по высоте контейнера (по умолчанию)
flex-startПрижать к верхнему краю
flex-endПрижать к нижнему краю
centerВертикально по центру
baselineПо базовой линии текста
✅
Идеальное центрирование: display: flex; justify-content: center; align-items: center — три строки CSS решают задачу, которая раньше требовала хаков с position или таблицами.

Урок 4 — flex-wrap и align-content

По умолчанию flex-элементы помещаются в одну строку, сжимаясь при нехватке места. flex-wrap: wrap разрешает перенос на новую строку. align-content управляет расположением строк по поперечной оси — аналог justify-content, но для строк в многострочном контейнере.

СвойствоЗначенияОписание
flex-wrapnowrap, wrap, wrap-reverseРазрешить перенос строк
align-contentflex-start, center, space-between...Выравнивание строк (при wrap)

Урок 5 — flex-grow, flex-shrink, flex-basis

Эти три свойства управляют поведением отдельного flex-элемента. flex-grow — насколько элемент растёт занимая свободное место. flex-shrink — насколько сжимается при нехватке места. flex-basis — начальный размер до распределения пространства. Шорткат flex: grow shrink basis задаёт их вместе.

СвойствоЗначение по умолчаниюОписание
flex-grow0Коэффициент роста (0 = не растёт)
flex-shrink1Коэффициент сжатия (1 = сжимается)
flex-basisautoБазовый размер элемента
flex: 1—Равно flex: 1 1 0 — равномерный рост
flex: auto—Равно flex: 1 1 auto
flex: none—Равно flex: 0 0 auto — не гибкий

Урок 6 — order и align-self

order позволяет изменять порядок отображения flex-элементов без изменения HTML. По умолчанию все элементы имеют order: 0; меньшие значения отображаются первыми. align-self переопределяет align-items для конкретного элемента.

СвойствоПрименяется кОписание
orderflex-элементЦелое число; меньше = раньше (по умолч. 0)
align-selfflex-элементПереопределяет align-items для этого элемента
💡
order и доступность: Свойство order меняет только визуальный порядок, но не порядок DOM. Скринридеры и навигация клавиатурой следуют порядку HTML, а не CSS. Используйте order осторожно.

Задание: Напишите HTML+CSS страницу с адаптивной сеткой карточек:
  • Контейнер карточек должен использовать display: flex и flex-wrap: wrap
