Селекторы

CSS-селекторы — это шаблоны, которые позволяют выбирать HTML-элементы для применения стилей. Зная селекторы, вы сможете точечно оформлять любые элементы страницы: по тегу, классу, идентификатору, атрибуту, состоянию или положению в дереве документа. В этом разделе вы освоите все основные виды CSS-селекторов от простых до мощных комбинаций.

Урок 1 — Простые селекторы

Простые селекторы выбирают элементы по одному критерию: имени тега, классу, идентификатору или универсальному символу *.

СелекторПримерОписание
Тегp { }Выбирает все элементы <p>
Класс.box { }Выбирает элементы с class="box"
ID#header { }Выбирает элемент с id="header"
Универсальный* { }Выбирает все элементы на странице
💡
ID vs Класс: ID должен быть уникальным на странице (один элемент), тогда как класс может использоваться у множества элементов одновременно.

Урок 2 — Комбинированные селекторы

Комбинированные селекторы описывают отношения между элементами в HTML-дереве. Они позволяют точно указать, какой именно элемент нужно выбрать в зависимости от его окружения.

КомбинаторСинтаксисОписание
Потомокdiv pВсе <p> внутри <div> (на любом уровне)
Дочернийdiv > pТолько прямые <p> внутри <div>
Соседнийh2 + pПервый <p> сразу после <h2>
Общий соседh2 ~ pВсе <p> на том же уровне после <h2>
✅
Совет: Используйте дочерний комбинатор > вместо пробела, когда нужно стилизовать только прямых потомков и не затрагивать глубоко вложенные элементы.

Урок 3 — Псевдоклассы

Псевдоклассы позволяют выбирать элементы на основе их состояния или положения в документе. Они записываются через двоеточие: :hover, :focus, :nth-child().

ПсевдоклассОписание
:hoverПри наведении курсора
:focusКогда элемент в фокусе (поле ввода)
:first-childПервый дочерний элемент
:last-childПоследний дочерний элемент
:nth-child(n)N-й дочерний элемент (можно: 2n, 2n+1, odd, even)
:not(selector)Все элементы кроме указанного
:checkedОтмеченный checkbox или radio
💡
:nth-child — мощный инструмент: :nth-child(odd) выбирает нечётные элементы, :nth-child(even) — чётные. Это удобно для создания «зебры» в таблицах.

Урок 4 — Псевдоэлементы

Псевдоэлементы обозначаются двойным двоеточием :: и позволяют стилизовать определённую часть элемента или добавлять контент, не изменяя HTML.

ПсевдоэлементОписание
::beforeВставляет содержимое перед содержимым элемента
::afterВставляет содержимое после содержимого элемента
::first-lineСтилизует первую строку текста
::first-letterСтилизует первую букву текста
::placeholderСтилизует текст-подсказку в поле ввода
::selectionСтилизует выделенный пользователем текст
⚠️
Важно: Псевдоэлементы ::before и ::after требуют свойство content (даже пустое content: ""), иначе они не отображаются.

Урок 5 — Селекторы атрибутов

Селекторы атрибутов позволяют выбирать элементы по наличию или значению HTML-атрибута. Они очень полезны для стилизации ссылок, полей форм и кастомных data-атрибутов.

СинтаксисОписание
[attr]Элементы с атрибутом attr (любое значение)
[attr="val"]Точное совпадение значения
[attr^="val"]Значение начинается с val
[attr$="val"]Значение заканчивается на val
[attr*="val"]Значение содержит подстроку val
[attr~="val"]Значение содержит слово val (через пробел)
✅
Практическое применение: Используйте a[href^="https"] для стилизации только внешних ссылок, а a[href$=".pdf"] для ссылок на PDF-документы.
