Единицы измерения

CSS предлагает богатый набор единиц для задания размеров: от абсолютных пикселей и сантиметров до гибких относительных единиц em, rem, процентов и единиц вьюпорта. Правильный выбор единицы измерения — основа адаптивной и масштабируемой вёрстки. В этом разделе вы изучите все ключевые единицы и узнаете, когда применять каждую из них.

Урок 1 — Абсолютные единицы

Абсолютные единицы измерения имеют фиксированный физический размер, не зависящий от контекста. Они подходят для печатных носителей, но в вебе чаще всего используются только пиксели px.

ЕдиницаОписаниеСоответствие
pxПиксель экрана1px = 1/96 дюйма (экранный)
ptТипографский пункт1pt = 1/72 дюйма ≈ 1.33px
cmСантиметр1cm = 37.8px
mmМиллиметр1mm = 3.78px
inДюйм1in = 96px
pcПика1pc = 16px
💡
Пиксель в вебе: Экранный пиксель (px) — это условная единица CSS, не всегда равная физическому пикселю экрана. На дисплеях Retina один CSS-пиксель соответствует 2–3 физическим пикселям.

Урок 2 — Относительные единицы em и rem

em — единица, относительная к размеру шрифта родительского элемента. rem — единица, относительная к размеру шрифта корневого элемента (<html>). Ключевое отличие: при вложении em умножается, а rem остаётся стабильным.

ЕдиницаОтносительна кОсобенность
emfont-size родителяКаскадно умножается при вложении
remfont-size <html>Всегда предсказуемый размер
✅
Совет: Используйте rem для размеров шрифтов и глобальных отступов — так проще управлять масштабом всего сайта, изменяя лишь font-size на html.

Урок 3 — Viewport единицы

Единицы вьюпорта задают размер как долю от видимой области браузера. Они незаменимы для создания полноэкранных секций, адаптивных заголовков и элементов, размер которых должен зависеть от размера окна.

ЕдиницаОписаниеПример
vw1% ширины вьюпорта50vw = половина ширины окна
vh1% высоты вьюпорта100vh = вся высота окна
vmin1% меньшей стороныКвадратные элементы по меньшей стороне
vmax1% большей стороныЭлемент по большей стороне экрана
💡
100vh и мобильные браузеры: На мобильных устройствах 100vh может включать высоту адресной строки. Для точного полного экрана используйте 100dvh (dynamic viewport height).

Урок 4 — Проценты %

Проценты — относительная единица, вычисляемая от родительского элемента. Для width и height — это размер родителя, для font-size — размер шрифта родителя, для padding и margin — ширина родительского блока.

Свойство% вычисляется отПример
widthширина родителя50% = половина родителя
heightвысота родителя (если задана)Требует высоты у родителя
font-sizefont-size родителя150% = 1.5× родитель
padding / marginширина родителя (даже вертикальные!)Трюк для aspect-ratio
⚠️
height: 100% не работает без родителя: Чтобы дочерний элемент занял 100% высоты, у родителя должна быть явно задана высота. Иначе браузер не знает, 100% от чего считать.

Урок 5 — Когда использовать какие единицы

Нет универсального ответа — каждая единица подходит для своей задачи. Зная правила применения, вы сможете строить масштабируемые, доступные и адаптивные интерфейсы.

ЕдиницаЛучше всего дляИзбегать для
pxРамки (border), тени, 1px деталиРазмеры шрифтов, отступы
remРазмеры шрифтов, глобальные отступыВнутренние компонентные отступы
emPadding/margin внутри компонентаГлобальные размеры (каскадный эффект)
%Ширина колонок, адаптивные блокиВысота (без явной высоты родителя)
vw / vhПолноэкранные секции, hero-блокиМелкие компонентные размеры
vmin / vmaxКвадратные элементы, иконкиТипографика (непредсказуемо)
✅
Золотое правило адаптивной вёрстки: px для декора, rem для типографики, % для раскладки, vh/vw для экранных секций.

Сделайте сами — Упражнение 6

Создайте адаптивный макет, используя разные единицы измерения

Задание: Напишите HTML+CSS страницу, в которой:
  • Задайте размер шрифта заголовка в rem (минимум 2rem)
  • Используйте em для внутренних отступов (padding) блоков
  • Ширину основного контейнера задайте в процентах (не более 80%)
  • Добавьте блок с высотой в vh (например, 50vh)
