Display и видимость

Свойство display — одно из самых важных в CSS. Оно определяет, как элемент участвует в потоке документа: занимает ли всю строку, встраивается ли в текст или полностью исчезает. Правильное значение display даёт точный контроль над расположением и видимостью любого элемента.

Урок 1 — display: block, inline, inline-block

Три базовых значения display определяют фундаментальное поведение элементов. block занимает всю ширину и создаёт перенос строки. inline встраивается в текст, игнорируя width и height. inline-block сочетает оба: встраивается в строку, но принимает размеры.

ЗначениеЗанимает строкуwidth / heightТипичные теги
blockВсю доступную ширину✅ Работаютdiv, p, h1–h6, section
inlineТолько по содержимому❌ Игнорируютсяspan, a, strong, em
inline-blockПо содержимому (встроен)✅ Работаютimg, button (CSS)

Урок 2 — display: none vs visibility: hidden

Оба способа скрывают элемент визуально, но по-разному влияют на раскладку. display: none полностью убирает элемент из потока — как будто его нет. visibility: hidden скрывает элемент, но оставляет занятое им место.

СвойствоЭлемент виден?Место в потоке?Доступен для AT?
display: none❌ Нет❌ Удалено❌ Нет
visibility: hidden❌ Нет✅ Сохранено❌ Нет
opacity: 0❌ Нет✅ Сохранено✅ Да
⚠️
display: none и доступность: Элементы с display: none невидимы для скринридеров. Для скрытия только визуально (с сохранением доступности) используйте класс visually-hidden с position: absolute; clip: rect(0,0,0,0).

Урок 3 — display: flex (введение)

display: flex превращает элемент во flex-контейнер, а его прямые дочерние элементы становятся flex-элементами. Это мощный инструмент для создания одномерных раскладок — строк или столбцов. Подробное изучение Flexbox — в следующем разделе!

💡
Flexbox отменяет float: Дочерние элементы flex-контейнера не нужно «флоатить» для горизонтальной раскладки. display: flex сам расставляет их в ряд и управляет выравниванием.

Урок 4 — display: grid (введение)

display: grid создаёт двумерную сетку из строк и столбцов. В отличие от Flexbox, Grid позволяет управлять одновременно горизонтальной и вертикальной осями. Подробное изучение CSS Grid — в разделе 11.

✅
Flex vs Grid: Используйте Flexbox для одномерных раскладок (меню в строку, карточки в ряд), а Grid — для двумерных сеток (весь макет страницы, фотогалереи).

Урок 5 — display: table и table-cell

CSS позволяет имитировать поведение HTML-таблиц без использования тегов <table>, <tr> и <td>. Это полезно для выравнивания данных в столбцы и для вертикального центрирования в контейнерах с неизвестной высотой.
