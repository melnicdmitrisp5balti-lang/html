Анимации (@keyframes)

CSS-анимации позволяют создавать сложные многошаговые эффекты без JavaScript. Ключевые кадры (@keyframes) описывают промежуточные состояния, а свойства animation управляют воспроизведением.

Урок 1 — @keyframes: from/to и проценты

Правило @keyframes описывает ключевые кадры анимации. Используйте from/to для двух состояний или проценты для нескольких.

СинтаксисОписание
@keyframes name { from{} to{} }Анимация от начального до конечного состояния
@keyframes name { 0% {} 50% {} 100% {} }Несколько ключевых кадров через проценты
💡
from/to vs проценты: from = 0%, to = 100%. Используйте проценты для нескольких ключевых кадров: @keyframes bounce { 0%{} 50%{} 100%{} }

Урок 2 — animation-name, animation-duration

animation-name указывает имя @keyframes, а animation-duration задаёт длительность одного цикла анимации.

✅
Регистр важен: Имя анимации чувствительно к регистру. slidein ≠ SlideIn.

Урок 3 — delay, iteration-count, direction

Управляйте задержкой, количеством повторений и направлением анимации с помощью соответствующих свойств.

СвойствоЗначенияОписание
animation-delay0s, 0.5sЗадержка перед стартом
animation-iteration-count3 / infiniteКоличество повторений
animation-directionnormal / reverse / alternate / alternate-reverseНаправление воспроизведения
💡
alternate: animation-direction: alternate создаёт эффект маятника — анимация идёт вперёд, потом назад.

Урок 4 — timing-function, fill-mode

animation-timing-function управляет скоростью кадров, а animation-fill-mode определяет, какие стили применяются до и после анимации.

СвойствоЗначенияОписание
animation-timing-functionease, linear, cubic-bezierКривая скорости
animation-fill-modenone / forwards / backwards / bothСтили вне анимации
✅
forwards: animation-fill-mode: forwards — элемент сохраняет стили последнего кадра после завершения анимации. Без этого он вернётся в исходное состояние.

Урок 5 — Shorthand и несколько анимаций

Сокращённое свойство animation задаёт все параметры в одну строку. Несколько анимаций перечисляются через запятую.

СинтаксисОписание
animation: name duration timing delay count direction fill-modeПорядок shorthand
animation: spin 2s linear infinite, pulse 1s ease-in-out infinite alternateНесколько анимаций через запятую
💡
Несколько анимаций: animation: spin 2s linear infinite, pulse 1s ease-in-out infinite alternate
