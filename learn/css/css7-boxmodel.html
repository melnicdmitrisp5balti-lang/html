Блочная модель

Каждый HTML-элемент в CSS представляется как прямоугольный блок с четырьмя слоями: содержимое, внутренние отступы (padding), рамка (border) и внешние отступы (margin). Понимание блочной модели — фундамент вёрстки. Без него невозможно точно управлять размерами и расположением элементов на странице.

Урок 1 — Модель блока

Блочная модель CSS состоит из четырёх областей, вложенных друг в друга: content (содержимое) → padding (внутренний отступ) → border (рамка) → margin (внешний отступ). Демо ниже визуально показывает каждый слой.

СлойСвойства CSSОписание
Contentwidth, heightОбласть с содержимым (текст, картинки)
Paddingpadding, padding-top и др.Прозрачный отступ внутри рамки
Borderborder, border-widthВидимая или невидимая рамка
Marginmargin, margin-top и др.Прозрачный отступ снаружи рамки
💡
Итоговый размер элемента по умолчанию: total width = content width + padding-left + padding-right + border-left + border-right + margin-left + margin-right

Урок 2 — width и height

Свойства width и height задают размер области содержимого. Значение auto (по умолчанию) означает: для ширины — растянуться на всего родителя, для высоты — подстроиться под содержимое. При переполнении работает свойство overflow.

СвойствоЗначенияОписание
widthauto, px, %, remШирина области содержимого
heightauto, px, vhВысота области содержимого
min-width / max-widthpx, %Ограничения по ширине
min-height / max-heightpx, vhОграничения по высоте
overflowvisible, hidden, scroll, autoПоведение при переполнении

Урок 3 — box-sizing: content-box vs border-box

По умолчанию (content-box) width задаёт только ширину содержимого, а padding и border добавляются сверху. С border-box width включает padding и border — итоговый элемент ровно указанного размера. Это гораздо удобнее!

box-sizingЧто включает widthИтоговая ширина
content-boxТолько contentwidth + padding + border
border-boxContent + padding + borderРовно width
✅
Лучшая практика: Большинство современных проектов добавляют *, *::before, *::after { box-sizing: border-box; } глобально — это делает вёрстку намного предсказуемее.

Урок 4 — Схлопывание margin (margin collapse)

Вертикальные внешние отступы (margin-top и margin-bottom) соседних блочных элементов схлопываются: вместо суммы двух отступов применяется один — наибольший. Это стандартное поведение CSS, которое часто удивляет новичков.

СитуацияСхлопываниеРешение
Соседние блокиmargin-bottom + margin-top → max()Используйте только один margin
Родитель и первый ребёнокmargin-top ребёнка «вытекает» наружуДобавьте padding или border родителю
Пустой блокmargin-top и margin-bottom схлопываютсяДобавьте padding или border
⚠️
Схлопывание не работает для: flex-контейнеров, grid-контейнеров, элементов с overflow не visible, float-элементов и абсолютно позиционированных элементов.

Урок 5 — border: стили и border-radius

Свойство border — шорткат для задания ширины, стиля и цвета рамки. Стиль рамки обязателен: без него рамка не отображается. border-radius скругляет углы и позволяет создавать круги, капсулы и другие фигуры.

СтильВидСинтаксис
solidСплошная линияborder: 2px solid #333
dashedПунктирнаяborder: 2px dashed #333
dottedТочечнаяborder: 2px dotted #333
doubleДвойнаяborder: 4px double #333
grooveВдавленнаяborder: 4px groove #999
ridgeВыпуклаяborder: 4px ridge #999
noneНет рамкиborder: none

Сделайте сами — Упражнение 7

Создайте карточку товара с правильной блочной моделью

Задание: Напишите HTML+CSS карточку товара, в которой:
  • Карточка должна иметь border (рамку) и border-radius (скруглённые углы)
  • Используйте padding для внутренних отступов (не менее 16px)
  • Задайте margin для отступов от других элементов
  • Примените box-sizing: border-box и задайте конкретную ширину карточки
