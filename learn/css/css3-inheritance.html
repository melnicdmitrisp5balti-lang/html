Наследование

Наследование в CSS — это механизм, при котором дочерние элементы автоматически получают значения некоторых CSS-свойств от родительских элементов. Понимание того, какие свойства наследуются, а какие нет, помогает писать компактный и предсказуемый CSS. В этом разделе вы также освоите ключевые слова inherit, initial и unset для явного управления наследованием.

Урок 1 — Что наследуется

Свойства, связанные с оформлением текста, как правило наследуются автоматически. Это означает, что достаточно задать их один раз на родителе — и все вложенные элементы получат эти значения без дополнительных объявлений.

СвойствоОписание
colorЦвет текста
font-familyГарнитура шрифта
font-sizeРазмер шрифта
font-weightЖирность шрифта
line-heightМежстрочный интервал
text-alignВыравнивание текста
text-transformРегистр текста
letter-spacingМежбуквенный интервал
list-styleМаркеры списка
cursorВид курсора
💡
Практическое применение: Задайте font-family и color на элементе body — все элементы страницы унаследуют эти значения, и вам не придётся повторять их для каждого тега отдельно.

Урок 2 — Что не наследуется

Свойства блочной модели и визуального оформления блоков не наследуются. Каждый элемент получает их начальное (браузерное) значение, и для его изменения нужно явно задать стиль для этого элемента.

СвойствоНачальное значение
margin0
padding0
bordernone
backgroundtransparent
width / heightauto
displayinline
positionstatic
overflowvisible
⚠️
Фон не наследуется! Если родитель имеет background: yellow, дочерние элементы будут прозрачными (transparent), просто «просвечивая» родительский фон. Это не наследование, а прозрачность.

Урок 3 — inherit, initial, unset

CSS предоставляет специальные ключевые слова для явного управления наследованием. Они работают как значение для любого CSS-свойства.

Ключевое словоОписание
inheritПринудительно наследовать значение от родителя
initialСбросить к начальному значению по спецификации CSS
unsetЕсли свойство наследуется — применить inherit, иначе — initial
revertСбросить к значению по умолчанию браузера (User Agent)
💡
all: unset — сокращение для сброса всех свойств одновременно. Удобно при стилизации компонентов изолированно от остального CSS страницы.

Урок 4 — Управление наследованием на практике

В реальных проектах наследование используется для задания единых стилей текста на уровне контейнера и точечного переопределения там, где нужно. Рассмотрим более сложный пример с карточкой товара.

✅
Стратегия: Задавайте шрифт, цвет и межстрочный интервал на body или крупном контейнере. Переопределяйте только то, что нужно изменить в конкретных дочерних элементах. Это сокращает количество CSS-правил и упрощает поддержку.

Продемонстрируйте работу наследования в CSS

Задание: Создайте HTML-страницу, которая наглядно показывает наследование:
  • Задать цвет и шрифт на родительском элементе и показать, что они унаследованы дочерними
  • Показать свойство, которое НЕ наследуется (border или background)
  • Использовать ключевое слово inherit для принудительного наследования
